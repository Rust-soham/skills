### Learning Implication

`SYNTHESIS`: An unfamiliar abstraction may feel like one opaque knot because
several responsibilities are compressed together. A useful FAFO curriculum can
vary or remove each responsibility independently, then recombine them.

Moving up the problem tree asks which primitive, invariant, or representation
would make several local difficulties disappear at once. This can produce a
threshold-style compression rather than another patch.

### Signals

- Learner separates concerns previously treated as one thing
- A broader invariant explains several local symptoms
- An enabling primitive makes the original task simpler
- The learner returns to complete the original task

### Failure Modes

- Decomposing recursively without testing
- Solving a broader problem unrelated to the immediate constraints
- Rebuilding infrastructure to avoid the target learning
- Mistaking scope expansion for depth

### Skill Implication

A future exercise generator may include three optional transformations:

- **Split**: identify independently variable responsibilities.
- **Elevate**: find the broader problem generating the local one.
- **Enable**: make a preparatory change, then perform the now-easier target
  change.

These are candidate curriculum operators, not yet universal mandatory steps.
