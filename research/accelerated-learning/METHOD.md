# Research Method

## Authority

1. The transcripts define the substance.
2. The user's current framing clarifies intent and source relationships.
3. Matt Pocock's skills define the desired shape: concise, behavioral,
   non-repetitive, and artifact-oriented.
4. External learning theory may name or test a mechanism later. It must not
   overwrite the corpus.

## Per-Source Extraction

Record:

- **Claims**: What the source says is true
- **Mechanisms**: Why the claimed effect occurs
- **Procedures**: Ordered actions an agent or learner can perform
- **Signals**: Evidence used to diagnose learner state or progress
- **Artifacts**: Durable state the system creates or updates
- **Failure modes**: Behaviors that weaken learning
- **Examples**: Concrete demonstrations of a general idea
- **Refinements**: Places where an earlier formulation becomes more precise
- **Tensions**: Claims that appear incompatible or context-dependent
- **Implications**: Possible system behavior, kept separate from source claims

Every substantial note must cite source line numbers.

## Synthesis Rules

- Preserve distinct mechanisms even when they produce the same outcome.
- Merge repeated wording only after checking whether later versions add a
  condition, boundary, sequence, or failure mode.
- Prefer later explicit refinements over earlier exploratory claims.
- Do not resolve genuine conflicts by averaging them.
- Distinguish universal learning behavior from domain-specific curriculum.
- Distinguish global learner context from local task and project context.
- Separate pedagogy from product UI, gamification, and implementation ideas.
- Treat memorable names as provisional until their definitions stabilize.

## Claim Labels

- `SOURCE`: Directly supported by a transcript
- `REFINEMENT`: A later source narrows or improves an earlier claim
- `SYNTHESIS`: Inferred by combining multiple sources
- `QUESTION`: Not resolvable from current evidence

## Completion Test

Research is ready for skill design only when:

1. Every source has an evidence note.
2. Every core concept links back to its sources.
3. Repetition has been collapsed without losing refinements.
4. Contradictions and unresolved choices are visible.
5. The system can be explained as inputs, state, loops, outputs, and boundaries.
