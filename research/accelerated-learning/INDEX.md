# Accelerated Learning Research

## Objective

Extract the complete learning system contained in `transcripts/`, then derive
the first concise, platform-neutral agent skills.

The transcripts are the source of truth. Existing skills in this repository are
reference shapes for compression, structure, and instruction style only.
