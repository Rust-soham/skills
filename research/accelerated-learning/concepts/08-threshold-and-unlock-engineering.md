# Threshold and Unlock Engineering

**Status:** Core at module scale

## Core Claim

An unlock is a model transition that reorganizes perception, compresses prior
facts, reveals new relations, and enables novel action. It often replaces a
locally useful but limiting model, not a simply false one.

## Module Shape

`baseline -> near boundary -> breaking experience -> minimum reframe -> novel
verification`

Define:

- Capability unlocked
- Limiting model and where it works
- New invariant or generative model
- Engineered experience that breaks the limit
- Artifact making the new model operational
- Generative gate
- Transfer target

## Mining Candidate Thresholds

When the domain graph does not already contain a trusted threshold:

1. Collect cases that differ on the surface.
2. Strip away incidental representation and context.
3. List the primitives each current explanation uses.
4. Ask what relation or constraint remains invariant across the cases.
5. Rotate the representation, abstraction level, or coordinate system.
6. Locate the plausible old model that cannot explain all cases at once.
7. Test whether the candidate invariant generates predictions in unseen cases.

This produces threshold hypotheses, not truth. Ground the domain claim and
verify changed learner capability before promoting one into the graph.

## Crossing Evidence

- Explain old and new models.
- Identify the old model's failure boundary.
- Detect old-model mistakes in unseen cases.
- Build, repair, or predict with the new model.
- Connect ideas that previously appeared separate.

## Liminal Evidence

Before a visible crossing, progress may appear as repeated near-misses, partial
connections, better failure localization, or the same model breaking across
multiple representations. Treat these as a pre-unlock cluster, not proof of
stagnation or crossing. During this liminal state, revisit the same invariant
through multiple representations without prematurely declaring failure or
supplying a verbal label as if it were a completed transition.

## Boundaries

- Supporting facts, retrieval, and fluency may sit inside or between thresholds.
- A dramatic feeling of insight is not crossing evidence.
- Threshold labels remain hypotheses until generative behavior changes.
