# Scaffolding Without Epistemic Dependency

**Status:** Core

## Core Claim

External tools can increase output while weakening internal navigability.
Therefore, successful orchestration and genuine understanding must be measured
separately.

## Dependency Tests

After scaffolded success, require one or more:

- Prediction without the scaffold
- Reconstruction of the causal model
- Judgment between alternatives
- Detection of an invalid result
- Recovery from a changed failure
- Transfer to a novel case
- A learner-generated probe

## Context Integrity

Distinguish:

- Retrieved learner evidence
- Current-session evidence
- User-supplied claims
- Agent inference

If prior context is unavailable, state that fact and reconstruct provisionally.
Never manufacture remembered history, learner confusion, or progress.

## Boundaries

- Scaffolding is not the enemy; untested dependency is.
- Independent verification need not mean memorizing retrievable details.
- Delegation is valid when the learner retains auditability and recovery.
