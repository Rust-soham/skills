# Real Project and Toy World Oscillation

**Status:** Core

## Core Claim

Real projects supply stakes, integration, authentic constraints, and meaningful
failures. Sandboxes supply causal clarity, safety, and one-variable
experiments. Accelerated learning needs deliberate movement between them.

## Move to a Sandbox When

- Several unfamiliar variables change together.
- Feedback is too slow or noisy.
- Failure is costly or irreversible.
- A missing primitive prevents a meaningful prediction.
- Setup or integration details dominate the target concept.

Before leaving, define the exact question, isolated variable, preserved real
constraints, ending evidence, and project action that resumes.

## Return When

- The learner predicts the baseline and one boundary case.
- The isolated result answers the declared question.
- The repaired model can be reapplied.
- A concrete project action is available.

## Boundaries

- A passing toy does not close the loop.
- Staying in the project is not virtuous when evidence is confounded.
- Moving to a sandbox must not become an unbounded escape from shipping.
