# Concept Library

Canonical cards for the 22 concepts deduplicated from the transcript corpus.
Use the cards for later design work; use `../sources/` when exact transcript
evidence or source-specific nuance is needed.

Status meanings:

- **Core:** repeatedly supported and necessary to the system
- **Contextual:** useful under particular missions or learner states
- **Candidate:** promising operator that still needs runtime validation
