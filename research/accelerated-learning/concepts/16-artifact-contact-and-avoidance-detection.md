# Artifact Contact and Avoidance Detection

**Status:** Contextual, important for build missions

## Core Claim

For learning while building, the artifact is a reality and behavior rail.
Learning should produce model evidence and return to concrete artifact progress.

## Legitimate Detour Test

A detour is legitimate when it has:

- A mission-relevant uncertainty
- New evidence, a model update, or reduced project risk as output
- A bounded return condition
- Actual return after the condition is met

Ask:

1. What uncertainty will this resolve?
2. What evidence ends it?
3. Which target action resumes?

## Avoidance Signal

Suspect avoidance only when a pattern repeats:

- Departure from declared target friction
- No relevant evidence, model change, or artifact progress
- No return condition
- Continued expansion after the blocker is clear

## Boundaries

- Research, planning, refactoring, rest, and confusion are not avoidance by
  activity type.
- Record observed behavior, not psychological labels.
- Persist broader patterns only with repeated evidence and learner consent.
